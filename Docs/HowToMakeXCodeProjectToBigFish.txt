* В папке солюшена нужно создать подпапку "XCodeProj"
* В настройках проекта MonoDevelop iPhoneBuild/Additional monotouch arguments добавить: "-xcode=${SolutionDir}/XCodeProj"
* Запустить билд, в результате в папке "XCodeProj" сгенерится [your_app].xcodeproj и много других файлов
* Скопировать в папку XCodeProj файл info.plist
* В info.plist установить CFBundleExecutable ${EXECUTABLE_NAME}
	(если этого не сделать, XCode при попытке загрузить билд в устройство промямлит, дескать, не могу найти бинарник,
	 и ты будешь трахатся вечно и гугл тебе не поможет)
* В XCode выбрать активную схему с устройством а не симулятором
* В Build Options:
	* выбрать Compiler for C/C++/Objective-C: LLVM GCC 4.2
	* Dead Code Stripping сделать No, иначе XCode вырежит все наши нативные библиотеки (libLemon.a, libBfg_iOS_SDK.a) и приложение будет молча падать
	* Из Valid Architectures удалить armv7s
	* Во вкладке "Other Linker Flags" содержимое "Any iOS SDK" добавить к содержимому "ARMv7 architecture | Any iOS SDK"
* В проект добавить следующие фрейемворки: Security SystemConfiguration StoreKit iAD GameKit (нужно для работы BigFishSDK)
* В ресурсы добавить: Data.iOS и бигфишевские файлы ресурсов
* Запустить и скрестить пальчики
